2.1

1、数据类型

文本型（字符串类型）：

  • 用’’表示，占两个字节，Unicode编码

布尔型：

  • 缺省为false
  • 取值只为两种 true或者 false
  • 注意：不可以用注意：不可以0或非0的整数替代true和false

整型：

  • 有固定表数范围和字段长度
    byte 1
类型 占用空间
byte 1字节
short 2字节
int 4字节
long 8字节
  • java整型是int类型 声明long型常量使用L
    • long k = 9562155L
    • long k = 9562155（错误）
    • 如果一个比较大的整数（超出int表数范围），Java不会自动把它当成long处理

浮点型：

单精度 :4个字节
双精度:8个字节
默认常量为double （双精度加D）
float f = 3.14f；

基本数据类型：

1、数字之间自由使用下划线
2、整型、浮点均可使用
dou pi =3.14_15_26

“引用”是指向一个对象在内存中的位置

例子： String s1=”a string”

2.2

标识符:

1、什么是标识符？
 - 用来表示变量名、类名、方法名、数组名和文件名
 - 有效的字符序列
2、命名规则
 - 字母、数字 _ $ 组合而成
 - 以字母、下划线或美元符号开头，不能以数字开头
 - 类名首字母大学，如String
 - 变量、方法及对象的首字母小写
 - 的、常量大写所有字母
 - Java包全部小写

什么是关键字

  • 是指被系统所保留使用的标识符：如print、int

2.3

1、什么是变量

  • 在程序的运行过程中数值可变
  • 先声明后使用

2、变量的声明

  • 用分号（英文状态下）结束
  • 语法：类型 变量名 =初始值，变量名 = 初始值
  • 如 int min=10;

3、作用域与生存周期

  • 程序块：被包括在一对{ }中

  • 一个程序块定义一个作用域

  • 如

    作用域：{

      }

4、常量

  • 程序运行过程中都不会改变
  • 与变量形式基本一致，前面加关键词final
  • 常量标识符全部用大写字母表示
  • 如：final double pi=3.14;
  • 十进制，如12；八进制要求以0开头，如012；十六进制以0x开头，如0x12

5、浮点常量

  • 十进制形式，必须包含小数点 ，如3.14 100.0
  • 科学计数法，如3.14e2
  • 如：final double MIN = 3.14E2D;

package fist;
import java.util.*;

public class InitVar &#123;
    public static void main(String[] args) &#123; 
          int a=0;
          float b=0F;
          double c=0D;
          System.out.println("从键盘输入一个整数：");
          Scanner buf=new Scanner(System.in);
          a = buf.nextInt(); //调用nextInt()方法，并将读入的数赋值给a
          System.out.print("你输入的整数是"+a);     
          System.out.println("从键盘输入一个单精度浮点数：");
          b=buf.nextFloat();//调用nextFloat()方法，并将读入的数赋给b 
          System.out.print("你输入的单精度浮点数是"+b);  
          System.out.println("从键盘输入一个双精度浮点数：");
          c=buf.nextDouble(); //调用nextDouble()，将读入的数赋给c
          System.out.println("你输入的双精度浮点数是"+c);
