排序与查询

numpy.sort(data, axis=1, kind=None, order=None)

参数得使用

  1. data代表你要排列的数组
  2. axis代表排序方向 0表示按行操作，1按列操作
  3. 排序方法，默认是快速排序
  4. 可自定义字段，按照某个属性进行排序
创建一维数组
n = np.array([[1,4],
             [3,2]])
n
'''
结果
array([[1, 4],
       [3, 2]])
'''
对一维数组进行排序
np.sort(a,axis=0) #按照行进行排序
'''
结果
array([[1, 2],
       [3, 4]])
'''
np.sort(a,axis=1) #按照列进行排序
'''
结果
array([[1, 4],
       [2, 3]])
'''
整体进行排序
np.sort(n,axis=None)
'''
结果
array([1, 2, 3, 4])
'''
使用argsort进行返回排序的值
创建数组
n1 = np.array([[1,4,8],
             [3,5,2]])
'''
结果
array([[1, 4, 8],
       [3, 5, 2]])
'''

对==列==进行排序

np.argsort(n1,axis=0)
'''
结果
array([[0, 0, 1],
       [1, 1, 0]], dtype=int64)
'''

对==行==进行排序

np.argsort(n1,axis=1)
'''
结果
array([[0, 1, 2],
       [2, 0, 1]], dtype=int64)
'''

argsort等，可以获得数组中最大值和最小值

通过where等方法，获得坐标
# 但n1大于5的数值，用坐标显示
n1 = np.array([[1,4,8],
             [3,5,2]])

n1 = np.where(n1>5)

'''
output
(array([0], dtype=int64), array([2], dtype=int64))
坐标（0，2）
'''
# 但n2大于3的数值，用坐标显示
n1 = np.array([[1,4,8],
            [3,5,2]])

n2 = np.where(n1>3)

'''
output
(array([0, 0, 1], dtype=int64), array([1, 2, 1], dtype=int64))
坐标（0，1），（0，2），（1，1）
'''

将上述的索引取出

n1[n2]
'''
output
array([4, 8, 5])
'''

若想获得数组最大（小）值的位置—>将数组转换为一个序列

np.argmax(n1,axis=None)
'''
output
2
'''

chisquare生成卡方分布

  1. 卡方分布的定义
    若n个相互独立的随机变量ξ₁、ξ₂、……、ξn，均服从标准正态分布（也称独立同分布于标准正态分布），则这n个服从标准正态分布的随机变量的平方和Q=∑i=1nξ2i

  2. 作用

    • 检验拟合优度：也就是说可以检验一组给定数据与指定分布的吻合程度。如：用它检验抽奖机收益的观察频数与我们所期望的吻合程度。
    • 检验两个变量的独立性：通过这个方法检查变量之间是否存在某种关系。

  3. ==卡方检验拟合优度案例==

    • 期望计算：期望频数=（观察频数之和（1000）） X （每种结果的概率） 如：-2：977=（0.977）X（1000）
    • 卡方分布：检验统计量来比较期望结果和实际结果之间的差别，然后得出观察频数极值的发生概率
    • 检验统计量：x2=∑(O−E)2E（O代表观察期望，E代表期望频数）

    即：对于概率分布的每一个概率，取期望频数和实际频数的差，求差的平方数，再除以期望频数，然后将所有结果相加。

生成3个二项分布

np.random.binomial(8,0.3,5) #0.3表示发生的概率

'''
output
array([6, 2, 4, 1, 8, 7, 3, 5])
'''
